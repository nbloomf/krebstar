The Kreb Editor

nbloomf

This is the user manual, design documentation, and source code of the krebedit text editor. As of this writing the project is very much Not Ready for Prime Time and supports exactly zero users. It is under rapid development though, and I hope for it to be “self-hosting” – featureful enough that I can develop krebedit in krebedit – very soon.
